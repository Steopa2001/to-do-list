# Todo List App – React + Tailwind CSS

Una semplice applicazione di todo list creata con **React** e **Tailwind CSS** per esercitarmi con componenti, stato e layout responsive.  

## Funzionalità principali

- Aggiungere nuove task tramite un input e un pulsante “Add”.
- Segnare le task come completate/non completate con una checkbox.
- Eliminare una singola task con un pulsante/icone.
- Salvare la lista delle task in `localStorage` così non si perdono alla ricarica della pagina.
- UI realizzata interamente con Tailwind CSS (layout, colori, ombre, hover, ecc.).

## Tecnologie utilizzate

- React
- Tailwind CSS per lo stile
- JavaScript (ES6+)
